# Gervacio-chat-app 

# server side
need to install

--for windows OS

npm i socket.oi express cors 

--for Macbook

sudo npm install -g nodemon --unsafe-perm

sudo npm i socket.oi express cors
# to run server side 
npm start

# client side
need to install
npm i socket.io-client 

to run client side

npm run dev
